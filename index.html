Python Features

1. Easy to Learn: Python has simple syntax similar to English.

2. Open Source: Python is free to use and distribute.

3. Cross-Platform Compatibility: Python runs on Windows, Mac, a and Linux.

4. Rich Library Support: Python has built-in libraries for various tasks like data analysis, machine learning, and web development.

5. Object-Oriented Programming (OOP)

Python supports OOP principles like encapsulation, inheritance, and polymorphism.
